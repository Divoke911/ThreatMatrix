## Overview
ThreatMatrix is an AI-powered Security Operations Center (SOC) dashboard. It enables security analysts to monitor alerts, manage incidents, and get AI-assisted threat analysis (alert explanations, MITRE ATT&CK mapping, recommended actions) from a single role-based web interface.

## Purpose
This is a final-year academic project built with synthetic/seeded data (no live log ingestion).

## Tech Stack
- Frontend: React
- Backend: Flask + SQLAlchemy
- Database: PostgreSQL
- Auth: JWT (Flask-JWT-Extended)
- Deployment: Docker + docker-compose
- AI: Server-side LLM API (e.g., Claude/OpenAI)
