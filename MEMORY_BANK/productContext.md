# Product Context

## Purpose
ThreatMatrix exists to simulate a real Security Operations Center (SOC) analyst workflow for demonstration and academic purposes.

## Core Workflow
1. **Alert**: The system flags a potential security threat based on seeded/synthetic data.
2. **Investigate**: Analysts review the alert details. The AI Analyst assists by providing explanations, mapping to MITRE ATT&CK, and suggesting actions.
3. **Incident**: Validated alerts are escalated to incidents for formal tracking.
4. **Resolve**: The incident is remediated and closed.
