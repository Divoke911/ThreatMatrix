# ThreatMatrix Tech Stack & Rules

## Tech Stack
- **Frontend**: React
- **Backend**: Flask + SQLAlchemy
- **Database**: PostgreSQL
- **Authentication**: JWT (Flask-JWT-Extended)
- **Containerization**: Docker + docker-compose
- **AI Integration**: Server-side LLM API (Claude/OpenAI)

## Rules
1. **Roles**: Three distinct user roles exist: Admin, Analyst, Viewer.
2. **AI Calls**: All AI interactions must occur on the backend (server-side). Never make LLM API calls directly from the frontend.
3. **Security**: Secrets and API keys must never be hardcoded. Use environment variables.
